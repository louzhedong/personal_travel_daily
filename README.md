# 个人旅游记录网站

一个专注于帮助用户记录和可视化展示国内外旅游经历的Web应用。

## 功能特点

- 🗺️ **地图展示**：使用高德地图API展示国内/国外地图，支持标记和信息窗口
- 🌏 **国内/国外切换**：轻松切换国内和国外旅游记录模式
- 👥 **多用户支持**：支持多个用户，通过不同颜色区分标记
- 📍 **省份/国家标记**：支持选择省份或国家进行标记
- 🏙️ **城市筛选**：根据选择的省份/国家筛选相关城市
- ✏️ **游玩描述**：添加游玩时间、体验描述、照片链接等信息
- 💾 **本地存储**：使用IndexedDB存储数据，无需后端服务
- 📤 **数据导入/导出**：支持数据的备份和恢复功能

## 技术栈

- **前端**：React 18 + TypeScript + Tailwind CSS + Vite
- **地图**：高德地图JavaScript API
- **状态管理**：Zustand
- **本地存储**：IndexedDB (使用idb库)
- **路由**：React Router DOM

## 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 配置高德地图API密钥

在 `src/components/MapComponent.tsx` 文件中，将 `your-amap-api-key` 替换为实际的高德地图API密钥。

### 3. 启动开发服务器

```bash
npm run dev
```

### 4. 构建生产版本

```bash
npm run build
```

## 使用指南

### 1. 添加用户

1. 在右侧控制面板的"用户管理"部分，输入用户名并选择颜色
2. 点击"添加用户"按钮
3. 可以通过下拉菜单切换当前用户

### 2. 添加旅游标记

1. 选择"国内"或"国外"模式
2. 点击"添加新标记"按钮
3. 选择省份或国家
4. 点击"保存标记"

### 3. 添加城市访问

1. 在地图上点击已添加的标记
2. 在右侧控制面板的"添加城市访问"部分，选择城市、填写访问日期和游玩描述
3. 点击"添加城市访问"按钮

### 4. 管理数据

- **导出数据**：点击"导出数据"按钮，将所有旅游记录导出为JSON文件
- **导入数据**：点击"导入数据"按钮，选择JSON文件导入数据
- **清空数据**：点击"清空数据"按钮，清空所有旅游记录

## 注意事项

- 本项目使用本地存储（IndexedDB）存储数据，数据仅保存在当前浏览器中
- 地图功能需要高德地图API密钥，请确保正确配置
- 本项目为纯前端应用，无需后端服务
