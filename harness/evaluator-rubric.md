## 本仓库校准点

- 若完整 `npm run test` 失败，必须判断是否属于已记录 blocker；不能直接宣称全部通过。
- UI 改动必须同时考虑视觉风格、响应式和遮挡问题。
- 文档类新增必须遵守用户指定位置，避免根目录膨胀。
